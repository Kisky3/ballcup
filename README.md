# Ballcup
A small game about ball cup / ReactNative

# Setup
1. setup ReactNative environment
2. install expo

```
yarn global add expo-cli
```
# Init
```
expo init ballcap
```

```
.
├── .expo
├── .expo-shared
├── .git
├── .gitignore
├── App.js
├── app.json
├── assets
│   ├── adaptive-icon.png

│   └── splash.png

├── package.json

```
# Run
```
yarn android
```
